# 📷 JCamera - Modern Camera App for Linux

A feature-rich desktop camera application built with PyQt5 and OpenCV.

## Features

- 📷 **Photo Capture** - Take photos with timer support
- 🎥 **Video Recording** - Record video with microphone audio
- 🖥 **Screen Recording** - Record screen with region selection
- ⚙ **Advanced Settings** - Resolution, FPS, codec, quality, filters
- 🎨 **Live Filters** - Grayscale, negative, edge detection, sepia
- ⏱ **Timer** - 3s/5s/10s/15s/30s countdown for photos
- 🎵 **Audio Support** - Select any PulseAudio input source

## Quick Install

```bash
git clone https://github.com/jsticks779/jcamera-for-linux.git
cd jcamera-for-linux
chmod +x install.sh
./install.sh
```

Then launch: `jcamera` or find "JCamera" in your app menu.

## Requirements

- Python 3.8+
- FFmpeg
- A camera (built-in or USB)

Installed automatically by the install script.
